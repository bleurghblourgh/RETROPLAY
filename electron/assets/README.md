# RETROPLAY Desktop App Assets

Place your app icons here:

- `icon.png` - Main app icon (512x512 recommended)
- `icon.ico` - Windows icon
- `icon.icns` - macOS icon
- `tray-icon.png` - System tray icon (16x16 or 32x32)

## Creating Icons

You can use tools like:
- https://icoconvert.com/ - Convert PNG to ICO
- https://cloudconvert.com/png-to-icns - Convert PNG to ICNS

## Default Icon

If no icons are provided, Electron will use its default icon.
