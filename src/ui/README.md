# UI

Desktop UI components for RetroPlay.

## Files

- `mainWindow.py` - Main application window
- `modernMainWindow.py` - Modern styled main window
- `loginScreen.py` - User login screen
- `modernLoginScreen.py` - Modern styled login screen
- `settingsMenu.py` - Settings configuration panel
- `modernSettingsMenu.py` - Modern styled settings menu
- `controlPanel.py` - Playback controls (play, pause, skip, volume)
- `vinylDisk.py` - Animated vinyl disk component
- `visualizer.py` - Audio visualizer component
- `albumCard.py` - Album display card
- `albumGrid.py` - Grid layout for albums
- `playlistCard.py` - Playlist display card
- `playlistPanel.py` - Playlist management panel
- `button.py` - Custom button component
- `textInput.py` - Custom text input component
- `fileDialog.py` - File selection dialog
- `dragDropZone.py` - Drag and drop file zone
- `logo.py` - Application logo component
