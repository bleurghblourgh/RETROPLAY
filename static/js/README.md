# JavaScript

Client-side JavaScript modules for RetroPlay.

## Files

- `app.js` - Main application entry point
- `core-functions.js` - Core utility functions
- `ai-chat.js` - AI chat interface functionality
- `profile.js` - User profile management
- `vinyl-visualizer.js` - Vinyl disk animation and audio visualizer
- `context-menu.js` - Right-click context menu handling
