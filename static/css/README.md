# CSS

Stylesheets for the RetroPlay web interface.

## Files

- `style.css` - Main application styles
- `themes.css` - Theme color definitions
- `retro-ui.css` - Retro-styled UI components
- `vinyl-panel.css` - Vinyl player panel styles
- `login-enhanced.css` - Enhanced login screen styles
- `layout-fix.css` - Layout corrections
- `layout-revert.css` - Layout revert styles
