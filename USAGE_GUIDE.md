# RETROPLAY Usage Guide

## Getting Started

### Login Screen

**First Time Users:**
1. Press `F1` to switch to Register mode
2. Enter username, email, and password
3. Press `TAB` to move between fields
4. Press `ENTER` to register
5. Switch back to Login mode with `F1`
6. Login with your credentials

**Returning Users:**
1. Enter username and password
2. Press `ENTER` to login

## Main Player Interface

### Keyboard Controls

| Key | Action |
|-----|--------|
| `SPACE` | Play/Pause current song |
| `O` | Open file dialog to select music |
| `ESC` | Open settings menu |
| `L` | Logout |
| `←` `→` | Navigate albums/playlists |
| `↑` `↓` | Navigate lists |
| `ENTER` | Select/Confirm |

### Mouse Controls

- Click on control buttons to play, pause, skip, shuffle, or repeat
- Click on playlists or albums to select them

## Features

### 1. Playing Music

**Load a Song:**
1. Press `O` to open file dialog
2. Navigate using arrow keys
3. Press `ENTER` on a directory to open it
4. Press `ENTER` on an audio file to load it
5. Song will automatically start playing

**Playback Controls:**
- **Play/Pause**: Click center button or press `SPACE`
- **Next Track**: Click right button
- **Previous Track**: Click left button
- **Shuffle**: Click shuffle button (randomizes queue)
- **Repeat**: Click repeat button (cycles: off → one → all)

### 2. Visual Elements

**Vinyl Disk:**
- Rotates when music is playing
- Speed syncs with playback
- Displays neon glow effects
- Shows current song artwork (center)

**Visualizer:**
- Real-time audio frequency bars
- Particle effects at peaks
- Color-coded by intensity
- Retro grid aesthetic

### 3. Themes

**Available Themes:**
1. **Synthwave** - Classic 80s neon aesthetic
2. **Tokyo Nights** - Neon-lit cityscape colors
3. **Cyberpunk Red** - Dark with red accents
4. **Vaporwave Pastels** - Soft pastel colors
5. **Matrix Green** - Terminal green theme

**Changing Themes:**
1. Press `ESC` to open settings
2. Use `↑` `↓` to select theme
3. Press `ENTER` to apply
4. Theme is saved to your profile

### 4. Library Management

**Adding Music:**
- Use `O` to browse and add individual songs
- Songs are automatically added to your library
- Metadata is extracted from audio files

**Organizing:**
- Songs are grouped by album
- Albums display as circular icons
- Navigate with arrow keys

### 5. Playlists

**Creating Playlists:**
- Playlists are created through the database
- Use playlist panel to view existing playlists

**Playing Playlists:**
1. Navigate to playlist panel
2. Select playlist with arrow keys
3. Press `ENTER` to play all songs in order

**Playlist Features:**
- Square disc box icons
- Shows song count
- Displays creation date

### 6. Albums

**Album View:**
- Circular icons for each album
- Grid layout (4 per row)
- Shows album name and artist

**Playing Albums:**
1. Navigate with arrow keys
2. Press `ENTER` to play entire album
3. Songs play in order

## Settings Menu

**Access:** Press `ESC` from main player

**Options:**
- **Theme Selection**: Change color scheme
- **User Info**: View logged-in username
- **Volume**: Adjust playback volume (future feature)

**Exit Settings:** Press `ESC` again

## Tips & Tricks

1. **Quick Play/Pause**: Press `SPACE` anytime
2. **Theme Preview**: Themes apply immediately when selected
3. **File Navigation**: Use `..` to go up one directory
4. **Logout**: Press `L` to return to login screen
5. **Visual Effects**: All animations sync with music playback

## Audio Format Support

Supported formats:
- MP3 (most common)
- WAV (uncompressed)
- OGG (open format)
- FLAC (lossless)
- M4A (Apple format)

## Troubleshooting

**No Sound:**
- Check system volume
- Verify audio file format is supported
- Ensure audio device is connected

**Slow Performance:**
- Close other applications
- Reduce particle effects
- Use lower resolution display

**Can't Find Music:**
- Navigate to your music folder using file dialog
- Ensure files have supported extensions
- Check file permissions
