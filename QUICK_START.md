## Installation & Running

```bash
cd RETROPLAY
pip install -r requirements.txt
python main.py
```

Browser opens automatically to http://localhost:5000

## First Time Setup

1. **Register Account**
   - Enter username, email, password
   - Click "Register"
   - Login with credentials

2. **Upload Music**
   - Click "Upload" tab
   - Drag & drop music files
   - Watch AI analysis (BPM, mood)
   - Files appear in Library

3. **Play Music**
   - Go to Library tab
   - Click any song to play
   - Use player controls at bottom

## Troubleshooting

**Port already in use?**
- Close other apps using port 5000
- Or change port in app.py

**Upload not working?**
- Check file format (MP3, WAV, OGG, FLAC, M4A)
- Ensure uploads/music folder exists

**Theme not changing?**
- Clear browser cache
- Refresh page

**Context menu not showing?**
- Make sure JavaScript is enabled
- Try different browser
